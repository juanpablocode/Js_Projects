Operadores Lógicos

Retornan True ou false, comparam uma ou duas condições

Operadores Relacionais

  1. < Sinal de Menor que x
  2. > Sinal de Maior que x
  3. <= Sinal de Menor ou Igual
  4. >= Sinal de Maior ou Igual
  5. == Sinal de Igual(Comparação)
  6. != Sinal de Não Igual(Diferente)

Operadores Lógicos

  1. ! Sinal de Negação
  2. && Sinal de Conjunção

    Necessário que ambas condições sejam identicas para retornar true

  3. || Sinal de Desjunção("ou" lógico)

    Necessario que apenas 1 das condições sejam verdadeira para retornar true
